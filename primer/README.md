## Installation

```bash
$ git init
$ npm i
$ npm run prepare
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Debugging
First open a debug terminal
```bash
# for the app
npm run start:debug

# for tests run the commands
npm run test

```

## Linting
